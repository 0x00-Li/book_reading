## schema 与数据类型优化



### 数据类型优化

1. 更小的通常更好
2. 简单就好
3. 尽量避免null
4. 实际细则
   1. 整数类型
   2. 字符和字符串
   3. blob和text
   4. date和timestamp及datetime
   5. 使用枚举类型代替字符串
   6. 特殊数据类型，如IP
